# Personal Dashboard

A single morning/evening personal cockpit for planning, project tracking, and daily reflection.

## What it is
Pulls together tasks, calendar, project progress, and personal rhythms into one unified view. Designed for offline/local use.

## Features (planned)
- **Daily Planning Assistant** — input energy level and available time, surfaces tasks filtered accordingly
- **Project Tracker** — live status of song cycle, Kostka curriculum units, autodidact book vignettes
- **Bike Training Log** — ride tracking, climb segments, progress toward goal ride
- **Examen Journal** — daily examination of conscience, customizable theological prompts, private and local
- **Session Log Viewer** — aggregated focus time from `focus-timer`

## Pulls from
- Todoist (tasks)
- Google Calendar (schedule)
- [`focus-timer`](https://github.com/youssefevanw/focus-timer) — session logs
- [`ear-training`](https://github.com/youssefevanw/ear-training) — practice progress
- [`learning-runtime`](https://github.com/youssefevanw/learning-runtime) — Kostka curriculum progress
- [`research-hub`](https://github.com/youssefevanw/research-hub) — reading progress
